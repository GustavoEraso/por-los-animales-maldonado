    Function handlePromiseToast

    • handlePromiseToast<T>(
          promise: Promise<T>,
          options: PromiseToastOptions,
      ): Promise<T>

      Handles promise-based operations with automatic toast state management and custom icons.

      Shows a loading toast while the promise is pending, then automatically updates to success or error based on the promise result. Each state displays the appropriate custom icon:

      • Pending state: No icon (can be customized by adding type to pending message)
      • Success state: Shows logo300.webp (organization logo)
      • Error state: Shows error.webp (error icon)

      Perfect for async operations like API calls, file uploads, database operations, and batch processing. Supports Promise.all and Promise.allSettled patterns for handling multiple concurrent operations.

      Type Parameters

      • T

      Parameters

      • promise: Promise<T>

        The promise to track and display status for (including Promise.all results).

      • options: PromiseToastOptions

        Configuration for the promise toast.

        • messages

          Messages for each promise state (pending, success, error).

        • position

          Toast position.

        • autoClose

          Auto close delay, or false to disable.

        • hideProgressBar

          Whether to hide the progress bar.

        • closeOnClick

          Whether clicking closes the toast.

        • pauseOnHover

          Whether hovering pauses auto close.

        • draggable

          Whether the toast is draggable.

        • theme

          Toast theme.

      Returns Promise<T>

      Returns the original promise for chaining.

      Example

      // Handle an animal save operation with custom icons
      const savePromise = saveAnimalToDatabase(animalData);
      handlePromiseToast(savePromise, {
        messages: {
          pending: {
            title: 'Saving Animal',
            text: 'Please wait while we save the animal data...'
          },
          success: {
            title: 'Animal Saved!',
            text: 'The animal profile has been successfully saved to the database.'
          },
          error: {
            title: 'Save Failed',
            text: 'Unable to save animal data. Please try again.'
          }
        }
      });
      // Result: Pending shows no icon, success shows logo300.webp, error shows error.webp

      // Handle multiple operations with Promise.all

            title: 'Animal Created',

          },
          error: {
            title: 'Save Failed',
            text: 'Failed to save animal data. Please try again.'
          }
        }
      });
