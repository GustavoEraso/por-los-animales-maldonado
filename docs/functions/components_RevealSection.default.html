      Animated reveal section component with GSAP scroll-triggered animations.

      Creates an engaging section with image and text content that animates into view as the user scrolls. Features responsive design with different animation triggers for mobile and desktop. Uses optimized GSAP imports with SplitText for line-by-line text animations, and includes optional call-to-action link.
